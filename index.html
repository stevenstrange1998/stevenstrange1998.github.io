2025–Present · Lead iOS Developer · Sole engineer

Faysics: glasses cut from a face scan

Prescription glasses, manufactured from an iPhone face scan. A factory machines real parts from my measurements, so the whole pipeline is built for millimeter accuracy.

  • Ported the Open3D C++ library to iOS/ARM64, cross-compiled, wrapped in Objective-C++, surfaced in Swift. RGB-D fusion, ICP registration, and pose tracking run entirely on device.
  • Depth frames stream from the sensor, fuse into a 3D face mesh, and measurements (interpupillary distance, face and nose width) are read directly from geometry.
  • Corrected systematic sensor error by aligning scans to a canonical reference and cancelling offsets between the ARKit and Vision coordinate systems.
  • Wrote the real-time AR try-on renderer: depth-buffer occlusion, physically based materials, head-pose smoothing at interactive frame rates.

2023 · Personal project

VisionType Pro: an iPhone keyboard for Vision Pro

A device-to-device input link over Bluetooth and Wi-Fi that lets the iPhone keyboard type on Apple Vision Pro, because pecking at a floating keyboard in mid-air gets old fast.

  • Built on MultipeerConnectivity: peer discovery, pairing, session state, and keystroke streaming handled across the full connection lifecycle.
  • Input latency kept low enough to feel native, so keystrokes appear on the headset as you type them.

2026 · Full-stack personal project

Number Canvas: a paint-by-numbers social network

Turn any photo into a paint-by-numbers canvas, then share it to an Instagram-style feed. The computer vision runs on the phone; the backend is serverless. Built end to end.

  • Conversion pipeline turns photos into paintable canvases with superpixel segmentation, semantic masks, and edge detection, all on device.
  • Full backend on Cloudflare Workers and D1: relational schema, auth, canvas storage, and feed generation, running serverless at the edge.
