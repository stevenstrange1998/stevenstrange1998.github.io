# stevenstrange1998.github.io

Personal portfolio of Steven Strange — software engineer working at the
hardware/software boundary (C++, Swift, on-device ML and 3D geometry).

Static site, no build step. Hand-built HTML/CSS/JS with no framework or
chart libraries.

- `index.html` — home / project case studies
- `research.html` — brain-tumor CNN research write-up with interactive visualizations
- `styles.css` — design system
- `main.js` — shared behavior (nav, scroll reveals, counters)
- `viz.js` — research-page visualizations
- `faysics.js` — Faysics pipeline overview
- `Assets/web/` — optimized video, poster, and PDF assets

Hosted with GitHub Pages at <https://stevenstrange1998.github.io/>.
